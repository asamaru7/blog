CocoaPods Usage

설치

[sudo] gem install cocoapods
pod setup

사용법

패키지 검색

http://cocoapods.org/

Podfile 설정

cd 프로젝트 폴더
vi Podfile


platform :ios, '6.0'

pod 'HZWebViewController', '~> 0.0.2'

Pod 설치

pod install

Pod 업데이트

pod update

실헹

프로젝트명.xcworkspace

프로젝트명.xcodeproj가 아닌 프로젝트명.xcworkspace로 프로젝트를 열어야 정상적으로 빌드 및 실행이 된다.
